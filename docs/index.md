# SDK 开发指南

欢迎使用 ZsiBot 二次开发 SDK。

## 快速开始

1. 选择机型：`ZSL-1` / `ZSL-1w` / 
2. 根据机型进入 `demo/<model>/` 目录运行示例
3. 阅读 [部署与运行](deploy.md) 完成集成
4. 查看 [架构与目录结构](architecture.md) 理解模块关系
5. 查看机型对应的api文档了解具体接口定义
